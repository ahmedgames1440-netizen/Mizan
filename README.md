# ميزان — تطبيق الجوال (أندرويد)

نسخة جوال من برنامج ميزان لتحليل نتائج الطلاب، بكل وظائف نسخة سطح المكتب:
رفع ملفات نور، تحليل تلقائي، رسوم بيانية، تصدير PDF، تقارير فردية للطلاب،
كشف الحالات الشاذة، تقرير المعلم، مقارنة فصلين، شهادات تكريم، وإعدادات
دائمة للمدرسة.

---

## ⚠️ مهم جداً — اقرأ هذا أولاً

بناء تطبيق أندرويد (APK) **لا يتم على ويندوز مباشرة** — الأداة المستخدمة
(Buildozer) تعمل فقط على Linux أو macOS. إذا جهازك ويندوز، عندك خياران:

### الخيار الأول (موصى به): WSL — Windows Subsystem for Linux
هذا يشغّل Linux حقيقي بداخل ويندوز، بدون الحاجة لجهاز ثاني:

1. افتح PowerShell **كمسؤول (Run as Administrator)**
2. نفّذ هذا الأمر وانتظر التثبيت:
   ```
   wsl --install
   ```
3. أعد تشغيل الجهاز
4. بعد إعادة التشغيل، يفتح تلقائيًا تطبيق Ubuntu — اتبع التعليمات لإنشاء
   اسم مستخدم وكلمة مرور لهذا النظام الجديد (مستقل عن حساب ويندوز)
5. الآن لديك Linux يعمل بداخل ويندوز. افتح تطبيق "Ubuntu" من قائمة ابدأ
   في أي وقت تريد العمل على المشروع

### الخيار الثاني: جهاز Linux فعلي أو جهاز Mac
لو عندك أي جهاز يشغّل Linux (أو حتى Mac)، تقدر تستخدمه مباشرة بدون WSL.

## ⚠️ تنبيه فني مهم بخصوص الرسوم البيانية

البرنامج يستخدم مكتبة **matplotlib** لرسم الرسوم البيانية (الأعمدة،
الدونات). هذي المكتبة **معروف عنها مشاكل توافق متفرقة** عند تجميعها
لأندرويد (مش كل الحالات، لكنها احتمال حقيقي موثّق من مطوّرين آخرين).

**لتخفيف هذا الخطر، صمّمت البرنامج بحيث:**
- لو نجح البناء وعملت الرسوم البيانية بشكل طبيعي على جهازك — ممتاز،
  كل شيء يعمل كما هو متوقع.
- لو حصلت مشكلة بمكتبة matplotlib تحديدًا أثناء التشغيل الفعلي على
  الجوال (نادر، لكن ممكن)، البرنامج **لن يتعطل بالكامل** — بل تظهر
  رسالة "تعذّر عرض الرسم البياني" بمكان الرسم فقط، وبقية الشاشة
  (الأرقام، الجداول، الألوان) تستمر بالعمل بشكل طبيعي.

إذا واجهت هذي الرسالة بكثرة، تواصل معي وأبني حلًا بديلًا (مثلاً رسوم
بيانية أبسط برسم مباشر بـ Kivy بدون matplotlib).

## 🚀 خطوات البناء

داخل نظام Linux أو WSL (Ubuntu):

1. افتح الـ Terminal (أو Ubuntu من WSL)
2. انتقل لمجلد المشروع بعد فك الضغط، مثلاً:
   ```
   cd /mnt/c/Users/<اسمك>/Desktop/mizan_mobile
   ```
   (لو رفعت المجلد لسطح المكتب بويندوز وتستخدم WSL)
3. شغّل سكربت البناء:
   ```
   chmod +x build_android.sh
   ./build_android.sh
   ```
4. انتظر — أول مرة يحمّل Android SDK + NDK (قد يستغرق 20-40 دقيقة حسب
   سرعة الإنترنت، والحجم الكلي حوالي 3-5 جيجابايت)
5. عند الانتهاء، الملف الجاهز يكون بمجلد:
   ```
   bin/mizan-1.0-arm64-v8a_armeabi-v7a-debug.apk
   ```

## 📲 تثبيت الـ APK على جوالك

1. انقل ملف الـ APK لجوالك (عبر كيبل USB، أو رفعه على Google Drive
   وتحميله من الجوال، أو أي طريقة تفضّلها)
2. على الجوال: افتح الملف من مدير الملفات
3. أندرويد سيطلب منك السماح بـ "تثبيت من مصادر غير معروفة" — وافق
   (هذا طبيعي لأي تطبيق غير من متجر Google Play)
4. اضغط "تثبيت"

## 🧪 إذا واجهت مشكلة

- **"buildozer: command not found"**: تأكد إنك فعّلت البيئة الافتراضية
  أولاً (`source venv/bin/activate`) قبل تشغيل أي أمر buildozer يدويًا.
- **توقف أثناء تحميل SDK/NDK**: تأكد من اتصال إنترنت مستقر، وأعد تشغيل
  `./build_android.sh` (يستكمل من حيث توقف غالبًا).
- **فشل بناء matplotlib**: راجع التنبيه أعلاه. جرّب أولاً إزالة
  `matplotlib` و`numpy` من سطر `requirements` بملف `buildozer.spec`
  وأعد البناء — البرنامج سيعمل بدون رسوم بيانية (بس بالجداول والأرقام)
  حتى نجد حل بديل.
- **أي رسالة خطأ أخرى**: انسخ آخر 30-40 سطر من الخطأ بالكامل وأرسلها لي.

## 📁 بنية المشروع (للمرجعية)

```
main.py              نقطة تشغيل التطبيق + نظام التنقل (Drawer)
theme.py              ألوان التطبيق (نفس ألوان نسخة سطح المكتب)
widgets.py            مكوّنات واجهة قابلة لإعادة الاستخدام (بطاقات، جداول)
core/                 محرك الاستخراج والتحليل وتصدير PDF (مطابق لسطح المكتب)
screens/              كل شاشة من شاشات التطبيق (9 شاشات)
assets/fonts/         الخطوط العربية المرفقة (مُصححة لدعم كل الرموز)
assets/icon.png       أيقونة التطبيق
buildozer.spec        إعدادات بناء APK
build_android.sh      سكربت البناء بضغطة واحدة
```

## ✨ المزايا المتوفرة (مطابقة لنسخة سطح المكتب)

- رفع ملفات نتائج نور وتحليلها تلقائيًا
- لوحة تحكم: معدل عام، توزيع تقديرات، مقارنة فصول
- تحليل المواد، طلاب يحتاجون دعم، كشف حالات شاذة
- تقرير معلم بمادته، مقارنة فصلين دراسيين
- بيانات الطلاب مع تقرير PDF فردي لكل طالب
- إعدادات مدرسة دائمة (اسم، مدير، شعار)
- تصدير تقرير PDF شامل وشهادات تكريم
- ألوان موحّدة للتقديرات بكل الشاشات والتقارير
